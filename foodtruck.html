Foodtruck

Dans cette SAE, j’ai réalisé mon tout premier site web en HTML/CSS avec un peu de JavaScript. Le projet consistait à créer un site vitrine pour un food truck, avec une page d’accueil, un menu et une section contact. J’ai utilisé JavaScript pour permettre à l’utilisateur de changer la langue du site.

Ce projet m’a permis de découvrir les bases du développement web et de structurer un site simple, responsive et fonctionnel.

Durée du projet : 12h

Languages utilisé : HTML, CSS, JS

Sur ce projet, j’ai été impliqué à plusieurs niveaux :

• Conception et intégration en HTML et CSS
• Ajout de fonctionnalités interactives avec JavaScript
• Organisation de la structure et du contenu pour une expérience utilisateur fluide

Ce projet m’a permis de consolider mes compétences en intégration web
